# 需求文档 — 周易运势

---

## 1. 项目概述

### 产品名称
**「周易运势」** — AI 占卜解惑助手

### 目标用户
- 对周易文化感兴趣的普通用户
- 遇到人生困惑想寻求方向的决策者
- 日常想查看运势宜忌的轻度用户

### 核心价值
> 用千年周易智慧 + AI 智能解读，帮用户看清当下、指导未来。

---

## 2. 功能需求

### 2.1 首页仪表盘 (P0)

| 编号 | 功能 | 说明 |
|------|------|------|
| FR-01 | 每日运势卡片 | 展示今日卦象、宜忌、幸运颜色、幸运数字 |
| FR-02 | 问答入口 | 输入框 + 发送按钮，用户可以输入问题 |
| FR-03 | 热门问题 | 预设 6 个常见问题快捷入口（财运/事业/感情等） |
| FR-04 | 快捷入口 | 卦象查询 + 随机占卜卡片入口 |

### 2.2 占卜问答 (P0)

| 编号 | 功能 | 说明 |
|------|------|------|
| FR-05 | 起卦动画 | 提交问题后显示铜钱动画，增强仪式感 |
| FR-06 | 卦象展示 | 显示本卦和变卦的六爻线条图 + 卦名 |
| FR-07 | AI 解读 | DeepSeek/Claude 生成个性化运势解读 |
| FR-08 | 解读打字机效果 | AI 文字以打字机效果逐字呈现 |

### 2.3 每日运势 (P0)

| 编号 | 功能 | 说明 |
|------|------|------|
| FR-09 | 每日卦象 | 基于日期种子确定当日卦象（当天固定） |
| FR-10 | AI 宜忌 | AI 根据卦象生成 3 条宜 + 3 条忌 |
| FR-11 | 幸运颜色 | 基于卦象 + 日期生成个性化幸运色 |
| FR-12 | 幸运数字 | 基于卦象 + 日期生成 1-99 幸运数字 |

### 2.4 搜索功能 (P1)

| 编号 | 功能 | 说明 |
|------|------|------|
| FR-13 | 卦名搜索 | 输入关键字搜索 64 卦 |
| FR-14 | 卦象详情 | 展示卦辞、彖传、象传、六爻爻辞 |
| FR-15 | 浏览列表 | 无搜索词时展示全部 64 卦列表 |

---

## 3. 不做清单 (MVP)

- ❌ 用户注册/登录系统
- ❌ 数据库持久化
- ❌ 分享卡片/截图
- ❌ 语音输入
- ❌ 小程序版本
- ❌ 黄历/农历集成

---

## 4. 验收标准

| 场景 | 验收条件 |
|------|----------|
| 打开首页 | 看到每日运势卡片（卦象/宜忌/幸运色/数字），输入框和热门问题 |
| 点击热门问题 | 跳转占卜页 → 铜钱动画 → 展示卦象 → AI 打字机解读 |
| 手动输入问题 | 同上流程，解读内容与问题相关 |
| 搜索"乾" | 返回乾卦等匹配结果，点击进入详情 |
| 卦象详情 | 展示完整卦辞、彖传、象传、六爻爻辞 |
| AI 不可用 | 降级显示基于卦辞的模板解读，不影响使用 |
