# 开发需求文档 — 可编辑图表工具

> 版本：v1.0 | 日期：2026-07-12 | 状态：开发中

---

## 1. 项目概述

### 1.1 项目名称
**可编辑图表工具**（Editable Chart Tool）

### 1.2 项目定位
一个纯浏览器端运行的数据可视化工具。用户无需安装任何软件，双击 HTML 文件即可在浏览器中使用。支持手动录入数据或上传文件，快速生成多种类型的图表，并自由编辑图表内容和样式。

### 1.3 目标用户
- 不懂编程的普通用户
- 需要快速制作图表的办公人员
- 学生、教师等需要数据可视化的群体

---

## 2. 功能需求

### 2.1 数据输入

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-01 | 手动输入 | 网页内嵌可编辑表格，用户逐格填写数据 | P0 |
| F-02 | 上传 CSV | 点击上传或拖拽 .csv 文件，自动解析到表格 | P1 |
| F-03 | 上传 Excel | 点击上传或拖拽 .xlsx/.xls 文件，自动解析到表格 | P1 |
| F-04 | 默认示例数据 | 首次打开时预填示例数据，方便用户快速体验 | P0 |
| F-05 | 添加/删除行 | 表格支持动态增减行 | P0 |
| F-06 | 添加/删除列 | 表格支持动态增减列 | P0 |

### 2.2 图表生成

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-07 | 柱状图 | Bar Chart，支持多系列对比 | P0 |
| F-08 | 折线图 | Line Chart，支持多系列趋势 | P0 |
| F-09 | 饼图 | Pie Chart，展示占比分布 | P0 |
| F-10 | 散点图 | Scatter Chart，展示两变量关系 | P1 |
| F-11 | 图表类型切换 | 点击切换图表类型，无需重新配置 | P0 |

### 2.3 图表编辑

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-12 | 修改数据 | 表格中修改数值 → 图表实时更新 | P0 |
| F-13 | 修改标题 | 自定义图表标题文字 | P0 |
| F-14 | 修改轴标签 | 自定义 X 轴、Y 轴名称 | P0 |
| F-15 | 配色切换 | 从预设配色方案中选择 | P1 |
| F-16 | 自定义颜色 | 通过颜色选择器自由指定颜色 | P2 |

### 2.4 导出

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-17 | 导出 PNG | 将当前图表导出为 PNG 图片下载 | P1 |

---

## 3. 非功能需求

| 编号 | 类别 | 要求 |
|------|------|------|
| NF-01 | 运行环境 | Chrome / Edge 浏览器（Chromium 内核） |
| NF-02 | 安装要求 | 零安装，双击 index.html 即用 |
| NF-03 | 加载速度 | 首屏加载 < 3 秒（含 CDN 资源） |
| NF-04 | 界面语言 | 中英混合（UI 中文，图表术语英文） |
| NF-05 | 数据隐私 | 所有数据仅在浏览器本地处理，不上传服务器 |
| NF-06 | 文件大小 | 上传文件限制 5MB，超出给出提示 |

## 4. 版本规划

| 版本 | 内容 | 预计 |
|------|------|------|
| v0.1 | 阶段 0-3：骨架 + UI + 数据表 + 柱状图 | MVP 最小可用 |
