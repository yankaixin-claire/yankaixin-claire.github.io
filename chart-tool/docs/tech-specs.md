# 技术规范 — 可编辑图表工具

> 版本：v1.0 | 日期：2026-07-12

---

## 1. 技术架构

### 1.1 整体架构

```
┌────────────────────────────────────────┐
│           index.html (单文件)           │
│  ┌──────────────────────────────────┐  │
│  │         HTML (结构层)             │  │
│  │   Header + Sidebar + Main Area   │  │
│  ├──────────────────────────────────┤  │
│  │         CSS (表现层)              │  │
│  │   内嵌样式，淡蓝色主题             │  │
│  ├──────────────────────────────────┤  │
│  │         JavaScript (逻辑层)       │  │
│  │   ┌──────┐ ┌──────┐ ┌──────┐    │  │
│  │   │Store │ │Table │ │Chart │    │  │
│  │   │数据核│ │表格  │ │图表  │    │  │
│  │   └──────┘ └──────┘ └──────┘    │  │
│  └──────────────────────────────────┘  │
├────────────────────────────────────────┤
│         CDN 外部依赖（4 个库）          │
│  Chart.js │ SheetJS │ PapaParse │      │
│  html2canvas                          │
└────────────────────────────────────────┘
```

### 1.2 设计模式：发布-订阅（简易版）

```
Data Store (单一数据源)
    │
    ├── 数据变化 → 通知订阅者
    │     ├── TableRenderer.render()
    │     └── ChartRenderer.render()
    │
    └── 所有修改通过 Store.set() 统一入口
```

## 2. 外部依赖

### 2.1 CDN 库清单

| 库名 | 版本 | CDN 地址 | 用途 | 大小(估) |
|------|------|----------|------|----------|
| Chart.js | 4.4.0 | `cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js` | 图表渲染 | ~60KB |
| SheetJS | 0.18.5 | `cdn.jsdelivr.net/npm/xlsx@0.18.5/dist/xlsx.full.min.js` | Excel 解析 | ~120KB |
| Papa Parse | 5.4.1 | `cdn.jsdelivr.net/npm/papaparse@5.4.1/papaparse.min.js` | CSV 解析 | ~20KB |
| html2canvas | 1.4.1 | `cdn.jsdelivr.net/npm/html2canvas@1.4.1/dist/html2canvas.min.js` | PNG 导出 | ~30KB |

**版本锁定原则**：所有 CDN 地址均锁定大版本号，避免自动升级导致的 API 不兼容。

### 2.2 不使用构建工具

- ❌ 不依赖 npm / yarn / pnpm
- ❌ 不依赖 webpack / vite / rollup
- ❌ 不依赖 Node.js 运行环境
- ✅ 浏览器原生 ES6+ 语法直接运行

## 3. 数据模型

### 3.1 Data Store 结构

```javascript
const store = {
  // 表格数据
  headers: ['月份', '销售额', '利润'],    // 列标题（第一列为 X 轴标签）
  rows: [
    ['1月', 120, 45],
    ['2月', 150, 55],
    ['3月', 180, 60],
  ],

  // 图表配置
  chartType: 'bar',     // 'bar' | 'line' | 'pie' | 'scatter'
  title: '月度销售数据',
  xLabel: '月份',
  yLabel: '金额（万元）',

  // 样式配置
  colorScheme: 'blue',  // 配色方案名称
  customColors: [],     // 自定义颜色数组

  // 订阅者列表
  _listeners: [],       // 数据变化时通知的回调函数
};
```

### 3.2 数据转换规则

| 图表类型 | X 轴数据 | Y 轴数据 | 特殊处理 |
|----------|----------|----------|----------|
| 柱状图 | `rows[][0]`（第一列） | `rows[][1..n]`（其余列） | 每列一组柱子 |
| 折线图 | 同上 | 同上 | 每列一条线 |
| 饼图 | `rows[][0]`（第一列） | `rows[][1]`（仅第二列） | 仅支持单系列 |
| 散点图 | `rows[][1]`（第二列） | `rows[][2]`（第三列） | 两列数值 |

## 5. 安全说明

- 所有数据处理完全在浏览器本地进行
- 不上传任何数据到远程服务器
- 不依赖第三方 API 调用
- CDN 加载的库文件均有 SRI (Subresource Integrity) 校验（后续添加）
